# ML_models

A collection of machine learning projects I built while learning data science.
Each notebook uses a real dataset, picks a model that actually fits the data, and walks through the full process — cleaning, EDA, training, and evaluation. The datasets are also shared for you to test.

---

## Stack

- Python 3.10 
- pandas, numpy
- scikit-learn
- matplotlib, seaborn

No paid tools or APIs. Everything runs locally or on Google Colab.

---

## What I was practicing

- Picking the right model based on the data, not just using whatever algorithm I just learned
- Handling missing values in ways that make sense for the domain (not always filling with median)
- Evaluating models properly — accuracy is not enough for imbalanced datasets
- Explaining results in plain terms, not just printing metrics
